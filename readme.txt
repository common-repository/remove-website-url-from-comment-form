 This is a very simple Plugin which removes Website url from Comment form and help you to cut down on spam. 
 
== Description ==

Spammers want to have a backlink from your site and the quick way to have is to comment on your posts and for site owner removing these spam comments are hectic. this plugin simply remove website url field so spammers are less likely to attack your site.

 

== Installation ==

 
1. Upload `remove-website-url-from-comment-form.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Thats it. Now Comment form will not show website url.
 
== Frequently Asked Questions ==
 
= Should i take my website backup before activating this plugin ? =
 
This plugin is real simlple and does not do anything which can break your site but its always safe to take a backup whenever you doing anything with plugins.
